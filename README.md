
## 🚀 Login Facebook API


### Mô tả

Chương trình này sử dụng LoginService để đăng nhập vào Facebook bằng tài khoản cung cấp. Sau khi đăng nhập thành công, chương trình sẽ trả về **`Cookie`** và **`Token`** của tài khoản.
